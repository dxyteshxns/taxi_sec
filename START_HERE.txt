⚡ QUICK LINUX SETUP

  On Linux server, extract this project and run:

    bash setup.sh

  That's it! The script will:
    ✓ Install PostgreSQL
    ✓ Create database: taxi_db_sec
    ✓ Setup Python environment
    ✓ Install all dependencies
    ✓ Run migrations
    ✓ Load sample data

  Then create admin and start server:

    source .venv/bin/activate
    python manage.py createsuperuser
    bash run.sh

💡 RECOMMENDED WORKFLOW

  1. On Windows:
     • Run: prepare_for_linux.ps1
     • This creates: taxi_service_linux.zip

  2. Transfer to Linux:
     • Use SCP, SFTP, or USB

  3. On Linux:
     • Extract: unzip taxi_service_linux.zip -d taxi_service
     • Setup: cd taxi_service && bash setup.sh
     • Create admin: python manage.py createsuperuser
     • Run: bash run.sh

  4. Access:
     • Open: http://YOUR_SERVER_IP:8000
